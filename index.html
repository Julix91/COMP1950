Beginnings

Beginning of August 2016 Julian completed the Lynda.com "Become a programmer" video series and enjoyed it enough to dive into an experiment. For one month he tested his aptitude for JavaScript using codecademy, codewars, and CodeCombat and learned much from Free Code Camp, The Odin Project, sitepoint and other tutorials, blogs and resources.

Links to the best sites

Google is my absolute favorite, because I depend on it to access the extension of my mind that is the internet. But it would be nothing without the awesome sites it can find.

  • W3schools is a web dev reference for beginners, that helped me when I started.
  • Mozilla Developer Network is the same thing with less hand-holding, and more depth.
  • Code Pen and similar sites are filled with great insirations and resources with MIT license (i.e. cite if you use it).
  • YouTube for video-casts and tutorials. If you struggle with the readily available procrastination opportunities there, try a designated site, like lynda.com or pluralsight.
